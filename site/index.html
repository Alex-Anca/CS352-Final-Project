Project Synopsis

Motivation

Inspired by Imogen Heap's MiMU gloves, Hands Off makes expressive music performance accessible with no hardware beyond a laptop and webcam. Anyone can manipulate a live music stream in real time using only their hands.

Overview

Hands Off is a gesture-controlled Infinite Jukebox. A song is analyzed into beats, clustered by harmonic and timbral similarity, and played back in a continuous stream. Hand gestures detected via MediaPipe reshape that stream live:

  • Left fist: loop the current beat
  • Right fist: eighth-note stutter
  • Left pinch + rotate: sweep a low-pass filter
  • Right pinch + rotate: control cluster-jump tightness
  • 1–4 fingers (hold 2s): select which beat cluster to stay in

Technical Approach

Audio is analyzed with librosa: chroma and MFCC features are extracted per beat, then clustered with Ward clustering. Beats are projected to 2D via PCA for a live scatter-plot display. The audio callback streams beat chunks with crossfade-on-jump and a continuously-swept Butterworth IIR low-pass filter. Gesture smoothing uses a 4-frame rolling mode buffer to suppress transient noise.

Related Work
